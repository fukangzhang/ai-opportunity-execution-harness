## 原因

- 官方要求至少使用一个 Microsoft 产品，实际生态重心偏 Microsoft Copilot Studio / M365 / Microsoft 低代码工作流。

- 8GB Windows 本地环境可以做 API/云端应用，但不应把时间投入到单一平台配置型 demo。
