## 为什么值得关注

- 当前主线方向匹配度最高：真实可工作的 Agent、清晰场景、演示视频和架构说明。
- 本地主要做产品、RAG、工具调用、流程编排和演示材料，8GB Windows 环境可控。
- 交付形式适合沉淀成后续 AI 应用赛通用模板。

## 交付物

- 至少一个使用 Microsoft 产品的 working agent。
- 不超过 5 分钟的 demo 视频。
- 架构概览：组件、数据流、集成点。
- 清晰的 use case 和目标用户说明。

## 前 48 小时目标

- 明确赛道：优先 Special Ops 或 Operative。
- 选定一个可演示业务场景。
- 做出最小可运行 agent demo，并记录输入、输出、失败样例。
- 补齐 `plan.md` 和 `evidence.md` 中的验收标准。

## 风险

- 需要至少使用一个 Microsoft 产品，必须尽早确认可访问账号和工具。
- Demo 视频和架构说明会影响评分，不能只堆功能。
- 可靠性与安全占评分，需要保留失败处理、日志和人工审核设计。

作为第一主线 active 比赛推进。后续开发只在本目录及对应项目仓库/演示材料目录中进行。
