# Agent Academy Hackathon 证据与验收

## 官方规则证据

- 官方页面： https://microsoft.github.io/agent-academy/events/hackathon/
- 核验日期：2026-05-14
- 关键事实：比赛窗口为 2026-05-12 到 2026-06-02；开放参赛；提交需要 working agent、5 分钟以内 demo 视频、架构概览和明确 use case。

## 最小可提交定义

- 一个用户能实际操作或观看完整演示的 Agent。
- 至少接入一个 Microsoft 产品。
- 能解释输入、处理流程、输出、失败处理和人工审核点。
- 有架构图或架构说明。
- 有不超过 5 分钟的 demo 脚本。
