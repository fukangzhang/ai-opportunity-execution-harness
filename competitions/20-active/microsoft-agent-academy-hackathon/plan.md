# Agent Academy Hackathon 执行计划

## 本轮目标

用 2 周左右完成一个可提交的 Agent 应用原型，交付 demo 视频、架构图和说明材料。

## 赛道倾向

优先级：

1. Special Ops：适合展示 MCP、外部系统集成、结构化输出、评测和安全模式。
2. Operative：适合展示多 Agent 协作、流程触发和业务动作。

## 候选作品方向

- 比赛调研与执行助手：读取比赛规则、检查资格、拆解交付物、生成 sprint plan、跟踪证据。
- RAG 证据包助手：把公开规则、来源、项目计划和实验记录整理成可审计提交包。
- 工作流交接助手：面向多 Codex/多分支并行协作，自动生成状态摘要和下一步边界。
