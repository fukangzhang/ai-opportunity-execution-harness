# Microsoft Agent Academy Hackathon

Source: https://microsoft.github.io/agent-academy/events/hackathon/

Window: 2026-05-12 to 2026-06-02.

Open to everyone except Microsoft employees and immediate family members.

The official material requires at least one Microsoft product. The practical center of gravity is Microsoft Copilot Studio, M365, and Microsoft low-code workflows.

Local constraint: Windows, 8GB GPU. API/cloud is feasible, but the project would become platform-config heavy instead of code-first.

Decision note: Use as reference only. Do not make it active because single-ecosystem lock-in reduces reuse value.
